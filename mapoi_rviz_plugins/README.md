# mapoi_rviz_plugins

`mapoi`のRvizプラグイン。

## mapoi_panel

Rvizの左下に出している。

- 地図の切替を行う
- 地図に応じた目的地を選択できる
- 選択した目的地に自己位置を修正する
- 選択した目的地に自律走行する
- このパネルで実行した自律走行を停止する

## poi_editor

POIの情報を表形式で編集できます。

## 開発方法

1. Qt creatorでuiファイルを作成
2. ヘッダファイル、ソースファイルを作成
